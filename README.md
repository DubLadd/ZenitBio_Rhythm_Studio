# ZenitBio_Rhythm_Studio


⚡ Core App Modules
We integrated several distinct, high-energy tools into a unified dark-cyber UI dashboard:
Rhythm Machine v4 (The Sequencer): A fully customizable step-sequencer that supports up to 64 steps for pattern creation. It handles master volume scaling, dynamic BPM adjustments, and multi-track expansion (up to 8 tracks).
Interactive Cross DJ Blender: A virtual live mixer featuring a functional crossfader. It visually and sonically blends two separate decks in real-time, pulling in your multi-frame graphic animations to glitch dynamically based on the fader's position.
Timeline Glitch Stitcher: A dedicated grid module inspired by your historical timeline fixer graphic. It functions as a manual loop-cutter, allowing you to instantly inject 1/4, 1/8, 1/16, or 1/32 stutter rates into the live playback stream.
Sub-Harmonic & Dialectal Labs: A sound-design testing bay equipped with custom sub-bass macro buttons (like the ultra-low Alley Chair rumble) and an unstable phase-distortion override toggle (Ć vs Č Razor).
🔊 Integrated Audio & Intelligence Engine
Web Audio API Synthesizer: The app doesn't rely on external static audio files for its drum hits. Instead, it features a built-in mathematical synthesizer engine. It generates raw sawtooth, sine, and square waveforms on the fly, feeding them through highly resonant low-pass filters and aggressive waveshaper distortion curves to mimic your heavy, live industrial techno basslines.
Live Audio Input & BPM Detection: Includes a custom file-uploader module. When you load an audio file into the system, it analyzes transient onset bursts to automatically calculate the track's native tempo and syncs the sequencer grid to it seamlessly.
Master Live Record Output: Features an inline audio-destination node that routes all synthesized signals and live crossfader movements directly into a high-quality MediaRecorder stream, allowing you to export your live sessions straight to a local downloadable file.
Embedded Gemini Co-Pilot: The entire interface links directly to an automated AI workspace chat module. It contextually reviews the files inside your repository array and can instantly write or refactor scripts, allowing you to patch modifications straight into the file system with a single click.
🎨 Visual & Structural Framework
CAPT-CHCE Sonar Waves: Features a live radar-canvas visualizer that dynamically generates concentric neonic shockwaves, expanding and pulsing in strict synchronization with every step trigger.
Full Hardware Responsiveness: Styled with custom Tailwind CSS layers and monospace layouts, making the panels shift smoothly on mobile screens, which is exactly why the app scales perfectly inside a container-to-APK shell on your phone.
It is a completely self-contained, offline-capable production studio mapped out to execute your raw, distorted live DJ energy on the go!
